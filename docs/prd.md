# GovOn v3 Product Requirements Document (PRD)

**버전**: 3.0.0 (지능형 행정 어시스턴트)  
**상태**: In Progress  
**대상**: 지자체 및 정부 부처 민원/행정 담당 공무원

---

## 1. Problem Statement
대한민국 공무원은 매년 수백만 건의 민원을 처리하며, 이에 따른 **답변 작성**과 **내부 보고용 공문서 작성**에 막대한 시간을 소요하고 있습니다. 
- **문제 1**: 민원 답변 시 복잡한 법령과 매뉴얼을 일일이 대조해야 함.
- **문제 2**: 내부 결재를 위한 공문서는 특유의 '개조식 문체'와 양식이 엄격하여 작성이 까다로움.
- **문제 3**: 기존 행정 시스템(온나라 등)은 폐쇄적이며 최신 AI 기술의 도움을 받기 어려움.

## 2. Goals & Success Metrics (KPI)
- **목표**: 기존 행정 시스템 옆에서 조력하는 **'지능형 사이드 패널'**을 통해 행정 효율성을 300% 향상시킨다.
- **성공 지표**:
  - 초안 생성 시간: 건당 20분 -> 1분 미만으로 단축.
  - 답변 정확도: 실제 법령/판례 근거 포함 비율 95% 이상.
  - 사용자 만족도: 복사/붙여넣기 편의성 만족도 4.5/5.0 달성.

## 3. Core Features (The "Agentic Side Panel")

### A. 에이전트 사이드 패널 (Interface)
- 기존 업무 화면을 방해하지 않는 슬림한 사이드바 UI.
- VS Code/Cursor와 유사한 대화형 인터페이스.
- **결과 카드**: 생성된 답변/공문을 즉시 확인하고 원클릭으로 클립보드에 복사.

### B. 에이전트 도구 상자 (Admin Skillset)
- **민원 해결 도구**: LBox 판례 및 사하구 조례 기반 최적의 답변 초안 생성.
- **공문서 작성 도구**: 행안부 공문서 데이터를 학습하여 보도자료, 정책보고서 등 개조식 문서 작성.
- **지식 검색 도구**: 수만 건의 국가중점데이터(ALIO, 법령)에서 실시간 행정 지침 검색.

### C. EXAONE v3 행정 지능 (Brain)
- 행안부 공문서 AI 학습데이터(말뭉치 + Q&A)로 파인튜닝된 전용 모델.
- 공무원 특유의 문체(개조식, 전문 용어) 완벽 구사.

## 4. User Journey
1. **상황**: 공무원이 온나라 시스템에서 민원 답변을 작성해야 함.
2. **호출**: 옆에 띄워진 GovOn 사이드 패널에 민원 내용을 붙여넣음.
3. **명령**: "이 건에 대해 법령 근거 포함해서 답변이랑 내부 보고용 공문 초안 뽑아줘."
4. **실행**: AI가 '민원 해결 도구'와 '공문 작성 도구'를 연속 호출하여 결과 생성.
5. **완료**: 공무원이 내용을 검수한 후 '복사' 버튼을 눌러 기존 시스템에 붙여넣음.

## 5. Non-Goals
- 기존 행정 시스템의 데이터베이스 직접 수정 (보안/권한 문제).
- 공무원의 최종 승인권 대체 (AI는 오직 '초안'만 작성).
